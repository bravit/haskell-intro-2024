# Lists

* List: head and tail
* Pattern matching over lists
* Functions over lists
* Polymorphism
* Type classes and bounded polymorphism
* Exploring lazy evaluations with `:sprint`

```
ghci> filter odd [1, 2, 3, 4]
[1,3]
ghci> :t filter
filter :: (a -> Bool) -> [a] -> [a]
ghci> map (\x -> x + 1) [1, 2, 3, 4]
[2,3,4,5]
ghci> :t map
map :: (a -> b) -> [a] -> [b]
ghci> :t quicksort 
quicksort :: Ord a => [a] -> [a]
ghci> :t length
length :: Foldable t => t a -> Int
```
